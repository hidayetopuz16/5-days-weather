# 🌤️ Modern Hava Durumu Uygulaması

Bu proje, kullanıcının hem manuel arama yaparak hem de GPS konumunu kullanarak anlık ve 5 günlük hava durumu tahminlerini görebileceği modern bir web uygulamasıdır.

## 🚀 [Canlı Uygulamayı Dene]([https://(https://hidayetopuz16.github.io/weather-app/)]

## 🛠️ Özellikler
* **🔍 Şehir Arama:** Dünyadaki herhangi bir şehrin hava durumunu sorgulayın.
* **📅 5 Günlük Tahmin:** Günlük sıcaklık ve hava durumu ikonlarıyla detaylı tahminler.
* **💎 Modern Tasarım:** Glassmorphism etkili, şık ve büyük butonlu arayüz.
